# Translations

This directory contains JSON translation files for all supported languages.

## Supported Languages

- `es/` - Spanish (Espa√±a - default)
- `en/` - English
- `fr/` - French
- `it/` - Italian
- `de/` - German

## Structure

Each language directory should contain:
- `common.json` - Common UI elements (buttons, navigation, etc.)
- `master.json` - Master admin interface translations
- `planner.json` - Wedding planner interface translations
- `admin.json` - Wedding admin interface translations
- `guest.json` - Guest RSVP page translations

## Conventions

- Use clear, descriptive keys (e.g., `button.submit`, `form.email.label`)
- Maintain consistency across all language files
- Use simple, clear language especially for guest-facing content
- Provide context in comments for ambiguous terms
