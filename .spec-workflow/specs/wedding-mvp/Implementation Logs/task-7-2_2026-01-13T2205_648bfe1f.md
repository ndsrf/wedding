- **Backend Endpoint:** GET /api/planner/stats
- **Data Flow:** Page mount → Fetch stats → Display in StatsCard components with wedding count, total guests, and RSVP completion percentage

#### Integration
- **Description:** Weddings list page fetches weddings from GET /api/planner/weddings and creates new weddings via POST /api/planner/weddings
- **Frontend Component:** PlannerWeddingsPage
- **Backend Endpoint:** GET /api/planner/weddings, POST /api/planner/weddings
- **Data Flow:** Page mount → Fetch weddings → Display WeddingCard for each → User clicks Create → WeddingForm modal → Submit → POST API → Refresh list

#### Integration
- **Description:** Wedding detail page fetches wedding details from GET /api/planner/weddings/:id and manages admins via POST/DELETE /api/planner/weddings/:id/admins
- **Frontend Component:** WeddingDetailPage
- **Backend Endpoint:** GET /api/planner/weddings/:id, POST /api/planner/weddings/:id/admins, DELETE /api/planner/weddings/:id/admins
- **Data Flow:** Page mount → Fetch wedding details → Display stats and admin list → User invites admin → AdminInviteForm → POST admin → Update admin list → User removes admin → DELETE admin → Update admin list
